# Todos

参考[Meteor Guide](http://guide.meteor.com/)的最佳实践和其对应的[todos](https://github.com/meteor/todos)项目来逐步完成一个基于Meteor＋React的demo项目.


## 特点

* 语言: ECMAScript2015 (ES6)
* 框架: Meteor 1.3+
* 前端: React 
* 路由: FlowRouter
* 布局: ReactLayout
* 测试: Mocha, Chai
* 样式: Less
* CI: [CircleCI](https://circleci.com)
* 代码检查: ESlint
* 数据库建模: SimpleSchema
